## Статус проекта: MVP готов

Сайт полностью функционален: каталог, корзина, оформление заказа, видеогалерея, отзывы, информационные страницы, админ-панель. Сервер запускается на `http://localhost:3000`.

- Skeleton-загрузка каталога
- Fade-up анимации карточек

- Расширенная мобильная адаптация

### Рефакторинг
- Создан common.js — общий модуль для всех страниц
- Убраны дублирующиеся inline-скрипты
- script.js полностью переписан

### Инфраструктура агентов
- AGENTS.md — точка входа
- AGENTS-ремесленник.md — инструкция разработчика

## Известные баги

1. `color-mix()` в CSS-переменных не работает в Safari < 16.2
2. Bento Grid первая карточка — при малом количестве преимуществ может выглядеть странно
3. Video-modal на YouTube-embed может не корректно определять thumbnail
4. Токен админа инвалидируется при перезапуске сервера
