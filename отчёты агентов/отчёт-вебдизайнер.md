## Предложенные и реализованные решения

### 1. Gradient text для логотипа
Логотип «TIMUR» получил градиентную заливку (accent2 → copper), что повышает узнаваемость бренда и соответствует тренду на «жидкие» градиенты 2025-2026.

### 2. Bento Grid для секции преимуществ
Первая карточка в сетке преимуществ занимает 2 колонки и 2 ряда — приём из Bento Layout тренда. Создаёт визуальную иерархию и ломает монотонность сетки.

### 3. Glassmorphism для модалки товара
Backdrop-filter: blur(12px) + полупрозрачный фон — мягкий, современный эффект, который не конфликтует с тёмной кожаной темой.

### 4. Skeleton-загрузка
Вместо пустого экрана при загрузке товаров показываются плейсхолдеры с shimmer-анимацией. Стандартный паттерн для улучшения perceived performance.

### 5. Custom scrollbar
Стилизованный скроллбар в тонах сайта — мелочь, которая подчёркивает внимание к деталям.

### 6. Микро-анимации
- Fade-up для карточек при скролле
- Cart bounce при добавлении в корзину
- Улучшенный hover для кнопок (transform + shadow)

### 7. Расширенная мобильная адаптация
- Video-modal: полноэкранный на 480px, адаптивный на 768px
- Bento Grid: отмена span на мобильных (возврат к 1fr)
- Detail modal: полноэкранный на маленьких экранах

## Рекомендации на будущее

1. **Dark/light toggle** — возможность переключения темы (тренд 2025+)
2. **Lottie-анимации** вместо GIF/emoji для иконок преимуществ
3. **Scroll-triggered video** — автоматическое воспроизведение видео в секции «О нас» при скролле
4. **3D-карточки товаров** — лёгкий perspective-эффект при наведении
5. **Magnetic buttons** — кнопки, притягивающиеся к курсору
6. **Cursor trail** — кожаная текстура/нитка за курсором (тематично!)
7. **Page transitions** — плавные переходы между страницами (view transitions API)
