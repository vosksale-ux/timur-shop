# TIMUR.SHOP — Интернет-магазин кожаных изделий

## О проекте
Магазин изделий из натуральной кожи ручной работы. Backend: Node.js + Express. Frontend: чистый HTML/CSS/JS. Хранение товаров — JSON-файл. Фото — multer в public/uploads/.

---

## Быстрый старт

1. Установить Node.js (https://nodejs.org)
2. Открыть терминал в папке проекта
3. `npm install`
4. `npm start`
5. Открыть http://localhost:3000
6. Админка: http://localhost:3000/admin.html

## Баги и проблемы (найдены при аудите)

### Критичные
1. **XSS в каталоге**: `renderProducts()` вставляет `p.name` и `p.desc` через innerHTML без экранирования. Если в названии товара будет HTML/JS — выполнится. Нужно написать функцию escapeHtml() и обернуть все пользовательские данные.
2. **XSS в корзине**: `cart.html` тоже вставляет `item.name` через innerHTML без экранирования.
3. **Нет авторизации админки**: Любой может открыть /admin.html и удалить/изменить товары.

### Средние
4. **Дублирование по имени в корзине**: `addToCart()` ищет товар по имени. Если у двух товаров одинаковое название — добавится к первому. Лучше искать по id.
5. **Идентификация корзины по имени, не по id**: localStorage хранит `{name, price, qty, image}` без id товара. При переименовании товара — корзина "отваливается".
6. **Удаление товара не удаляет фото с диска**: DELETE /api/products/:id удаляет товар из JSON, но не удаляет файлы фото из uploads/. Только удаление отдельного фото через /images чистит файл.
7. **Отсутствует валидация типов**: PUT/POST не проверяют что price — число, name — строка. Можно отправить отрицательную цену.
8. **Race condition**: Два одновременных запроса на запись могут перезаписать данные, т.к. readProducts/writeProducts не лочат файл.

### Мелкие
9. **Бургер-меню дублируется**: Код toggleMenu() повторяется в script.js, cart.html, admin.html, delivery.html, payment.html, about.html, reviews.html — 7 раз. Лучше вынести в общий файл.
10. **Обработчик nav-links тоже дублируется**: Блок querySelectorAll + addEventListener повторяется на каждой странице.
11. **Счётчик корзины не синхронизируется между вкладками**: Если открыть 2 вкладки и в одной добавить товар — во второй счётчик не обновится.
12. **Старый footer**: cart.html, delivery.html, payment.html, about.html, reviews.html используют простой `<footer>` вместо нового `.main-footer` с 4 колонками.
13. **Нет favicon**: Отсутствует иконка сайта.
14. **Нет 404 страницы**: При переходе на несуществующий URL — стандартный ответ Express.
15. **Нет сортировки товаров**: В каталоге нет сортировки по цене/названию.
16. **Нет поиска**: Нет поля поиска по товарам.
17. **Форма отзыва не сохраняется**: Отзыв отправляется в WhatsApp, но не сохраняется на сайте.
18. **Нет кнопки "Вверх"**: На длинных страницах нет кнопки прокрутки наверх.
19. **reviews.html: форма не очищается** после отправки.
20. **Год в copyright**: Везде стоит 2024, сейчас 2026.
