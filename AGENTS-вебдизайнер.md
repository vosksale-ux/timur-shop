## Дизайн-система TIMUR.SHOP

### Цветовая палитра (тёмная кожа)
```
--bg: #1a1410          основной фон
--bg2: #231c15         вторичный фон
--card: #2a2118        карточки
--text: #f0e6d8        основной текст
--text2: #c4b49e       вторичный текст
--accent: #c8843c      акцент (медь)
--accent2: #e6a55a     светлый акцент
--copper: #b87333      медь
--stitch: #5a4a38      строчка (декор)
--border: #3d3228      рамки
--danger: #c0392b      ошибки/удаление
```

### Правило 60-30-10 (для проекта)
```
60% — --bg / --bg2 (тёмно-коричневый фон)
30% --card / --border (карточки, секции)
10% — --accent / --accent2 / --copper (кнопки CTA, иконки, ссылки)
```

### Типографика
- **Шрифт:** Montserrat (веса: 400, 600, 700, 800, 900)
- **Заголовки:** clamp() для адаптивности
- **Текст:** line-height: 1.6+

### Компоненты
- Карточки товаров с каруселью изображений (Swiper-подобная)
- Модальные окна для детального просмотра
- Навигация: фиксированный header + бургер на мобильных
- Toast-уведомления
- Категории с переключением

## Правила

1. Предлагать решения, а не только критиковать
2. Опираться на базу знаний — давать ссылки на конкретные техники
3. Учитывать стек проекта (Vanilla JS, без React/Vue, без сборщиков)
4. Не предлагать тяжёлые библиотеки без необходимости
5. Проверять адаптивность всех предложений
6. Соблюдать CSS-переменные из `:root`
7. Не добавлять зависимости без запроса Владельца
8. Учитывать стиль кода проекта: `var`, `function`, без стрелочных
9. Русский язык для всех текстов
